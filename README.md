# VIdeoPlayer

Плеер видео с субтитрами
С помощью библиотеки OpenCV реализовать плеер с субтитрами.
Загружается видеофайл.
Выбирается файл субтитров формата SRT.
Проигрывается файл, поверх видео должны выводиться субтитры.

# Модули

VideoPlayer — Оркестратор (TODO)
Главный класс, координирующий работу всех компонентов.
```
load_video(filepath) — загрузить видеофайл
load_subtitles(filepath) — загрузить файл субтитров
run() — запустить цикл воспроизведения
handle_command(command) — обработать команду пользователя
```

PlayerContext — Состояние плеера
Хранит текущее состояние: пауза, включение субтитров.
```
is_paused() — проверить, на паузе ли плеер
toggle_pause() — переключить паузу
is_subtitles_enabled() — проверить, включены ли субтитры
toggle_subtitles() — переключить отображение субтитров
```

OpenCVVideoSource — Источник видео
Реализация интерфейса VideoSource на базе OpenCV.
```
open(filepath) — открыть видеофайл
is_opened() — проверить успешность открытия
read_frame() — прочитать следующий кадр
get_video_info() — получить метаданные (FPS, разрешение, длительность)
get_current_time_sec() — получить текущее время воспроизведения
seek(time_sec) — перемотать на указанное время
```

OpenCVRenderer — Отрисовка
Реализация интерфейса Renderer для отображения видео и субтитров.
```
render(frame, subtitle, context) — отрисовать кадр с субтитрами
is_window_closed() — проверить, закрыто ли окно
release() — освободить ресурсы окна
```

SrtSubtitleProvider — Провайдер субтитров
Парсинг и поиск субтитров формата SRT.
```
load(stream) — загрузить субтитры из потока
get_line(time_sec) — получить текст субтитра для указанного времени
```

ConsoleUserInterface — Пользовательский ввод (TODO)
Обработка команд с клавиатуры.
```
get_command() — получить следующую команду от пользователя
```

# Контейны

SubtitleCue — Блок субтитров
Представляет один субтитр с временем начала, конца и текстом.
```
get_id() — получить ID
get_start_time_sec() — время начала в секундах
get_end_time_sec() — время конца в секундах
get_text() — получить текст
is_active_at(time_sec) — проверить активность в момент времени
```

VideoInfo — Метаданные видео
Хранит информацию о видеофайле.
```
fps — кадров в секунду
width, height — разрешение
total_frames — общее количество кадров
duration_sec — длительность в секундах
```

PlayerCommand — Команды
```
Play, Pause, SeekForward, SeekBackward, ToggleSubtitles, Exit, None
```
